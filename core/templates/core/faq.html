How it works?
Each living human can register to recieve a daily income of 100 Utope. This is the only way the currency can come into existence. Utope derives its scarcity from the fact that each living human can register for at most one account.

How do the random links get chosen?
Using a distributed commit and reveal RNG. Registered user can generate a random value and commit to the hash of this value. After a fixed time interval they then reveal the value. The RNG combines these values to create a random value to choose the links.
